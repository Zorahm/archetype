# Archetype Mod — План промптов

## Стратегия разделения

### 🔴 Opus 4.6 (сложная архитектура, 3 промпта)

| # | Файл | Что делает | Почему Opus |
|---|------|-----------|-------------|
| 1 | `OPUS_01_SCAFFOLD.md` | Gradle, Architectury, структура проекта, платформенные абстракции | Мультилоадерная конфигурация — хрупкая, ошибка = проект не собирается |

| 3 | `OPUS_03_NETWORK.md` | Пакеты, синхронизация, серверная валидация | Безопасность + корректность сетевого протокола |

## Порядок выполнения

```
Фаза 1 (Opus):  #1 → #2 → #3
Фаза 2 (MM):    #4, #5, #8 (параллельно)
Фаза 3 (MM):    #6, #7 (зависят от #5, #8)
Фаза 4 (MM):    #9, #13, #14 (параллельно)
Фаза 5 (MM):    #10 → #11, #12 (параллельно)
Фаза 6 (MM):    #15 (финал)
```

## Как использовать промпты

1. Начинай с Opus промптов по порядку (#1 → #2 → #3)
2. Результат каждого Opus-промпта копируй в контекст следующего
3. Для MiniMax-промптов: всегда вкладывай в контекст результаты Opus #1–#3
4. В каждом промпте есть секция `## Контекст из предыдущих шагов` — вставляй туда реальный код
